Download CosmoQuest for Windows

Cosmoquest is a two-dimensional platformer featuring original artwork and concepts. Players can select from four diverse character options, and work their way through the puzzles of each level.

This project was completed over the course of one semester, and involved milestone planning, frequent reports and updates, and research of the GDevelop5 gaming engine. This engine featured an efficient user interface which converted all data to a JSON file to be interpreted by a browser.

The extensibility of this game was put in place during coding, and can be easily manipulated in the future to provide new character abilities, longer levels, and more challenges.
